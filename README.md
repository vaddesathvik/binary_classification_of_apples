🧠 Agentic AI Tutor

An AI-powered personalized learning assistant that acts as your agentic tutor, capable of reasoning, planning, and adapting to your study style.
Built using OpenAI’s GPT models and LangChain, it helps users:

📚 Understand complex topics through interactive, step-by-step explanations

🧾 Summarize PDFs and research papers with key takeaways

🗂️ Generate study notes, quizzes, and reports automatically

🤖 Coordinate multiple agentic workflows for specialized learning tasks

🎯 Adapt dynamically to user goals and progress

Tech Stack:
OpenAI API • LangChain • Python • PyMuPDF • Streamlit(frontend)
