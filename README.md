📌 Overview

Built an image-based binary classification system to classify apples (e.g., Good vs Defective) using machine learning.

🚀 Features
Image preprocessing using OpenCV
Feature extraction and model training
Models: XGBoost, LightGBM
Achieved ~90% accuracy
🛠️ Tech Stack

Python, OpenCV, NumPy, Scikit-learn, XGBoost, LightGBM
